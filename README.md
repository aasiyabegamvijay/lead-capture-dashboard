#  Lead Capture Dashboard with Webhook Integration

A full-stack MERN web application for capturing leads and triggering automation workflows via webhook integration. Built with clean UI/UX, responsive design, and production-style backend architecture.


##  Features

- Lead capture form with validation
- Lead listing dashboard with search
- Lead detail view
- Webhook trigger on lead creation
- Webhook success/failure tracking
- Dark mode toggle
- Responsive UI (Mobile + Desktop)
- Clean API architecture


##  Tech Stack

### Frontend
- React (Vite)
- React Router DOM
- Axios
- Custom CSS (Modern UI + Dark Mode)

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- Axios (Webhook calls)
- dotenv

### Tools
- MongoDB Atlas
- Webhook.site (Testing webhook endpoint)
- GitHub
